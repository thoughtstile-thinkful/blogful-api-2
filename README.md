### Installed Files

Used for migrations:

```
npm i postgrator-cli@3.2.0 -D
```

Knex Driver:

```
npm i pg
```

Migration Commands:

```
npm run migrate -- 0
npm run migrate
```

Seeding database command:

```
psql -U dunder_mifflin -d blogful -f ./seeds/seed.blogful_articles.sql
```

Install knex:

```
npm i knex
```
